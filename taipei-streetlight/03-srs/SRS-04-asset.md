## 4.4 資產異動

### SRS-04-020 資產異動管理

**來源**：§4.4

#### User Story

> 身為 **GOV_STAFF**，我可提出資產加帳/除帳/變更申請，經審核流程通過後更新資產資料。

#### 主要流程

1. 承辦人提出異動申請（單筆或批次），選擇類型：加帳/除帳/變更
2. 填寫異動原因、附件、應備文件
3. 提交後進入 ASSET_CHANGE 工作流審核
4. 審核通過後自動更新資產資料
5. 所有異動留歷程紀錄（device_events）

#### 驗收準則

| AC ID | 驗收條件 |
|-------|---------|
| AC-04-020-1 | 支援單筆與批次加帳、除帳、變更 |
| AC-04-020-2 | 異動須經機關審核流程（ASSET_CHANGE workflow） |
| AC-04-020-3 | 審核通過後自動更新資產資料 |
| AC-04-020-4 | 所有異動留歷程紀錄（who/when/what/before/after） |
| AC-04-020-5 | 可附加應備文件 |

#### 狀態：⚠️ 資產 CRUD 已建，工作流 ASSET_CHANGE 類型已定義，串接待開發
