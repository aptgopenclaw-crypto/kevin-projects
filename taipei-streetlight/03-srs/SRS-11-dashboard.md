## SRS-11-001 儀表板自訂

**來源**：§11-(1)

### User Story

> 身為 **GOV_MGR / GOV_CHIEF**，我可自訂儀表板布局、組件風格及配色，選擇顯示項目與圖表類型。

### 驗收準則

| AC ID | 驗收條件 |
|-------|---------|
| AC-11-001-1 | 得標廠商提供至少 3 種儀表板布局/配色方案供機關選定 |
| AC-11-001-2 | 可自訂顯示項目（從已定義的組件中選擇） |
| AC-11-001-3 | 可選擇圖表類型（折線圖/柱狀圖/圓餅圖/數字卡片/地圖） |
| AC-11-001-4 | 支援分頁管理（多個儀表板頁面） |
| AC-11-001-5 | 組件可拖曳調整位置與尺寸 |
| AC-11-001-6 | 儀表板配置可儲存與載入 |

### 技術設計

- **圖表引擎**：Apache ECharts
- **佈局引擎**：vue-grid-layout（拖曳式格線）
- **資料聚合**：後端預計算 + Redis 快取，避免即時重查詢
- **即時更新**：WebSocket / SSE 推送關鍵指標變動

### 狀態：❌ 未開始

| AC-11-002-6 | 報修案件來源數量及比例 |
| AC-11-002-7 | 故障分類數量及比例 |
| AC-11-002-8 | 智能路燈及智慧電表訊息回報即時性 |
| AC-11-002-9 | 本年開關箱安全檢查/改善累積數量及比例 |
| AC-11-002-10 | 本年燈桿安全檢查累積數量及比例 |
| AC-11-002-11 | 列管架空管線數量、改善數量及比例 |
| AC-11-002-12 | 可區分個別廠商維護情形 |

### 資料來源

| 指標 | 來源模組 | 查詢方式 |
|------|---------|---------|
| 報修/完修數量 | §5 repair_tickets | COUNT + status filter |
| 待修數量 | §5 repair_tickets | status NOT IN (CLOSED) |
| 平均修復時間 | §5 repair_tickets | AVG(completed_at - created_at) |
| 照明妥善率 | §4 devices + §5 | active_devices / total_devices |
| 來源統計 | §5 repair_tickets.source | GROUP BY source |
| 故障分類 | §5 repair_tickets.fault_category | GROUP BY fault_category |
| 安全檢查 | §5 inspection_records | GROUP BY result |
