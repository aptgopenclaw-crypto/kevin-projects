# 11 維護儀表板（Dashboard）

> 來源：需求 11-(1) ~ 11-(2) ｜預估工時：30

## 功能清單

### 11-1 自訂佈局
- 得標廠商提供儀表板布局、組件風格及配色供機關選定
- 機關可依據需求自訂：
  - 顯示項目
  - 數據視圖（圖表類型）
  - 分頁
  - 圖示尺寸

### 11-2 視覺化數據呈現

以圖表、圖形、表格和指標等視覺化形式呈現、監控和分析各項動態及靜態維護數據。

#### A. 維護情形
- 一般路燈及智慧路燈：
  - 本年累積報修數量、完修數量及比例
  - 本月累積報修數量、完修數量及比例
  - 目前待修數量
  - 平均修復時間
  - 照明妥善率
  - 報修案件來源數量及比例
  - 故障分類數量及比例
- 智能路燈及智慧電表訊息回報即時性
- 本年開關箱安全檢查、改善累積數量及比例
- 本年燈桿安全檢查累積數量及比例
- 列管架空管線數量、改善數量及比例
- 可區分**個別廠商**維護情形

#### B. 停電及施工資訊
- 台電停電區域
- 施工拆桿無法提供照明區域

#### C. 故障熱區
- 故障集中區域視覺化呈現

#### D. 績效指標達成度
- 各項績效指標達成度
- 依賴：績效管理模組（09）

#### E. 路燈盞數
- 路燈總盞數
- 機關與各廠商各別維護路燈盞數及比例
- 一般路燈與智能路燈盞數及比例
- 路燈、園燈及其他照明設施盞數及比例
- 各類光源盞數及比例

#### F. 開關箱數量
- 開關箱總數量
- 共桿開關箱、獨立開關箱及其他類型數量及比例
- 分界點位置達 1.5 公尺與未達數量及比例
- 智慧電表數量

#### G. 附掛物統計
- 附掛物總數量
- 附掛物分類數量及比例
- 各單位附掛數量及比例

#### H. 電費支出
- 近 3 年各年月電費及差異比較
- 包燈及表燈電費及比例

#### I. 其他
- 智慧電表狀態

## 技術考量

| 項目 | 方案 |
|------|------|
| 圖表引擎 | ECharts / Apache ECharts（已使用） |
| 佈局引擎 | 拖曳式 Grid Layout（如 vue-grid-layout） |
| 數據聚合 | 後端定期預計算 + 快取（避免即時大量查詢） |
| 即時更新 | WebSocket / SSE 推送關鍵指標變化 |
| 匯出 | 報表截圖（PDF）或數據匯出（Excel） |
