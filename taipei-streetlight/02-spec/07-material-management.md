# 07 材料管理

> 來源：需求 7-(1) ~ 7-(2) ｜預估工時：20

## 功能清單

### 7-1 機關使用端

#### A. 基本資料管理
- 庫別、材料規格、供應/承攬廠商、使用者
- 新增、修改、刪除

#### B. 採購管理
- 採購詳細表製作
- 驗收文件管理
- 採購材料規格資料建檔

#### C. 收料管理
- 交貨單管理
- 進貨作業
- 轉庫作業

#### D. 庫存管理
- 庫存總覽查詢
- 庫存表單
- 庫存盤點
- **安全庫存量預警提示**
- 庫存調整作業

#### E. 出料管理
- 領料申請
- 領料表單
- 出料作業

#### F. 報核管理
- 費用支出表報
- 廢品處理（繳庫）

### 7-2 得標廠商端
- 建置廠商所需之材料管理功能
- 由路燈管理平台**直接**查詢（非外部連結）：
  - 各案得標廠商換裝維護之所有材料規格
  - 各月（年）使用數量
  - 安全庫存量
- 統計分析：
  - 材料使用壽命
  - 品質分析
  - 備存需求

## 核心實體

| 實體 | 說明 |
|------|------|
| `warehouses` | 庫別 |
| `material_specs` | 材料規格 |
| `suppliers` | 供應/承攬廠商 |
| `purchase_orders` | 採購單 |
| `purchase_items` | 採購明細 |
| `receiving_records` | 收料紀錄 |
| `inventory` | 庫存（庫別 + 材料） |
| `inventory_adjustments` | 庫存調整/盤點 |
| `issue_requests` | 領料申請 |
| `issue_records` | 出料紀錄 |
| `disposal_records` | 廢品處理紀錄 |
