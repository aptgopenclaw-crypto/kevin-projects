# 09 績效管理

> 來源：需求 9-(1) ~ 9-(2) ｜預估工時：20

## 功能清單

### 9-1 績效指標自動計算

#### A. 指標管理
- 績效指標評核項目之新增、修改、刪除
- 計算公式及條件設定

#### B. 數據匯入
- 可依績效指標計算需求
- 單筆或整批匯入計分所需數據資料

#### C. 計算與呈現
- 依平台紀錄或輸入資料計算：
  - 各期本案各項績效指標服務水準
  - 績效分數
- 介面呈現績效統計分數
- 匯出應檢附文件之報表：
  - 加、扣點資料
  - 估驗計價

#### D. 鎖定功能
- 一經鎖定則固定該期績效計分內容
- 避免因資料更新造成計分變動

### 9-2 公式動態調整
- 實際績效計算公式與需求依機關執行決議更新及調整

## 核心實體

| 實體 | 說明 |
|------|------|
| `kpi_definitions` | 績效指標定義（名稱、公式、條件） |
| `kpi_periods` | 績效計算期別 |
| `kpi_data_imports` | 匯入之計分數據 |
| `kpi_results` | 各期績效計算結果 |
| `kpi_locks` | 期別鎖定紀錄 |

## 設計考量

- 公式引擎：需支援動態公式解析（如 SpEL、JavaScript 引擎或自訂 DSL）
- 鎖定後不可逆：需有明確的解鎖審核流程
- 資料來源：跨模組（報修完修數、智能路燈妥善率、巡查完成率等）
