# 10 行動作業應用程式 APP

> 來源：需求 10-(1) ~ 10-(5) ｜預估工時：20

## 功能清單

### 10-1 資產普查
- 依機關需求校對及記錄：
  - 路燈、分電箱、電源回路、架空管線及人手孔等設備資產資訊
- 可於平台 Web 介面查看並編輯其餘相關資料

### 10-2 維護履歷記錄
- 記錄換裝維護前、中、後照片
- 日期、時間及地理資訊（GPS 坐標位置）
- 可於平台 Web 介面查看並編輯

### 10-3 巡查記錄
- 依機關需求進行：
  - 燈桿安全檢查
  - 分電箱安全檢查
  - 架空管線巡檢
- 可於平台 Web 介面查看並編輯

### 10-4 拍攝功能與驗證
- 具備拍攝維護、巡查過程照片、影片功能
- **定位限制**：得要求定位於資產周邊 **50 公尺**內進行操作
- 權限設定：
  - 強制開啟行動裝置之位置標籤
  - 日期、時間浮水印
- 非 APP 拍攝之照片/影片匯入時：
  - 比對 **EXIF** 資訊確認為原始未修改版本
  - 位置標籤及日期、時間吻合
  - 經機關同意後始得異動

### 10-5 離線支援
- 現場無法連通網路時
- 接通網路後**自動續傳**維護履歷及照片

## 技術考量

| 項目 | 方案 |
|------|------|
| 開發框架 | React Native / Flutter / 原生（Android + iOS） |
| 離線儲存 | SQLite 本地暫存 + 背景自動同步 |
| 定位驗證 | GPS 坐標與資產坐標距離計算 ≤ 50m |
| 照片浮水印 | 拍攝時即時嵌入（不可後製） |
| EXIF 驗證 | 後端 API 解析 EXIF metadata 比對 |
| 上傳機制 | 分片上傳 + 斷點續傳 |

## 與 Web 平台整合

- APP 與 Web 共用相同 RESTful API
- 認證機制：JWT（與 Web 端一致）
- 資料即時同步至平台資料庫
