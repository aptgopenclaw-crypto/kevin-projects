# 台北市路燈公開資料與組織架構

## 一、公開資料來源

### 臺北市路燈位置分布圖

- **平台**：臺北市資料大平臺 (data.taipei)
- **資料集網址**：<https://data.taipei/dataset/detail?id=262e80cf-579c-4bfb-ba73-31621bc84616>
- **提供機關**：工務局公園處
- **下載連結**：
  - CSV：<https://tppkl.blob.core.windows.net/blobfs/TaipeiLight.csv>
  - JSON：<https://tppkl.blob.core.windows.net/blobfs/TaipeiLight.json>
- **資料筆數**：145,919 筆
- **欄位說明**：

| 欄位名稱 | 說明 |
|-----------|------|
| SerialNumber | 序號 |
| Dist | 行政區 |
| Quantity | 數量 |
| LightKind1~5 | 燈具種類（最多5種） |
| LightWatt1~5 | 燈具瓦數（最多5種） |
| LightHeight | 燈桿高度 |
| LightYear | 燈具年份 |
| TWD97X | TWD97 X 座標 |
| TWD97Y | TWD97 Y 座標 |
| UpdDate | 資料更新日期 |

> **注意**：座標系統為 TWD97（台灣大地基準 1997），非 WGS84 經緯度。如需轉換可使用 pyproj 等工具。

### 下設 18 個單位

#### ⭐ 路燈直接相關單位

| 單位 | 業務職掌 |
|------|----------|
| **路燈科** | 路燈、公園照明與水電設備等新建工程之勘測、規劃、設計、施工、估驗、計價、驗收作業及結算等事項 |
| **路燈工程隊** | 路燈與照明設施之管理、維護與工程之規劃、設計、施工、估驗及驗收等事項 |

#### 其他業務單位

| 單位 | 業務職掌 |
|------|----------|
| 園藝科 | 公園、綠地、廣場、兒童遊樂場等新建工程之勘測、規劃及設計 |
| 工務科 | 工程發包、簽約與土木、綠化、遊憩設施等工程之施工、估驗、計價、驗收及結算 |
| 品管及職安科 | 各類工程施工品質查核管理、勞工安全衛生業務之計畫督導、考核、訓練及管理 |
| 工程配合科 | 公園用地之取得，產權變更、移轉、囑託登記暨建物與地上物之協調、補償及拆遷 |
| 園藝工程隊 | 綠地、廣場與行道樹之管理及維護施工 |

#### 公園管理所（依行政區分工）

| 單位 | 管轄範圍 |
|------|----------|
| 園藝管理所 | 花卉繁殖栽培、花卉展覽、士林官邸與相關園區 |
| 陽明山公園管理所 | 士林、北投區公園 |
| 青年公園管理所 | 萬華、中正、大安區公園 |
| 圓山公園管理所 | 松山、中山、大同區公園 |
| 南港公園管理所 | 南港、信義、文山區公園 |
| 花卉試驗中心 | 觀賞花木試驗研究推廣、內湖區及陽明山區公園 |

#### 幕僚單位

| 單位 | 業務職掌 |
|------|----------|
| 資訊室 | 資訊作業設計、管理及推廣 |
| 秘書室 | 文書、檔案、出納、總務、財產管理與法制、公關、研考等 |
| 會計室 | 歲計、會計及統計 |
| 人事室 | 人事管理 |
| 政風室 | 政風事項 |

## 五、備註

- 路燈業務由公園路燈工程管理處統一管轄，其中「路燈科」負責新建規劃設計，「路燈工程隊」負責日常維護管理
- data.taipei 上無結構化的組織架構資料集，以上資訊整理自公園處官網
- 資料查詢日期：2026-04-25
