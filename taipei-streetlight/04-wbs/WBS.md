## 4. 依賴關係

```
Phase 1 ──→ Phase 2 ──→ Phase 4
  │                        ↑
  │            Phase 3 ────┘
  │
  └──→ Phase 5（整合驗證，依賴 Phase 1~4 全部）
         │
         ├──→ Phase 6（IoT，依賴 Phase 1 設備表 + Phase 5 事件流）
         │
         ├──→ Phase 7（績效/APP/儀表板，依賴 Phase 5 完整資料流）
         │
         └──→ Phase 8（NFR，依賴 Phase 5~7 功能完成後進行壓測/安全測試）
```

**關鍵路徑**：Phase 1 → Phase 2 → Phase 4 → Phase 5 → Phase 7 → Phase 8
