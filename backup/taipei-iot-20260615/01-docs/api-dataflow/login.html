📥 API Data Flow — POST /v1/noauth/login

Controller: AuthController#login ・ Service: AuthServiceImpl#login ・ Provider: LocalAuthProvider#authenticate（AuthType=LOCAL）
Rate limit: 10/min per IP ・ Captcha 必要 ・ Refresh token 寫入 HttpOnly Cookie

4. 成功路徑下的四個分支

A. 需強制改密碼
PasswordExpiryStatus ≠ OK
  • 簽 passwordChangeToken（短效）
  • 不寫 user_session
  • 不發 refresh_token cookie
  • 回應 passwordChangeRequired=true
B. SUPER_ADMIN
users.is_super_admin = true
  • access token scope = PLATFORM
  • tenant_id = null（不綁場域）
  • 權限 = ROLE_SUPER_ADMIN ∩ tenant_id IS NULL
  • 寫 user_session（tenant_id=null）
C. 單一 tenant
activeMappings.size() == 1
  • access token scope = TENANT
  • roles / dept / dataScope / permissions 全綁定
  • 寫 user_session
  • needsSelection=false
D. 多個 tenant
activeMappings.size() > 1
  • 只發 temporaryToken
  • 不寫 user_session（待 select-tenant）
  • 回 tenantOptions[] 給前端挑
  • needsSelection=true
